📖 Lesson: How to Print in Go

In Go, when you want to show something on the screen, you use fmt.Println(). Whatever you put inside the brackets with quotes will appear as output.

package main

import "fmt"

func main() {
    fmt.Println("anything you want to print")
}

👆 Use this same structure for every step. Just change what is inside fmt.Println().
