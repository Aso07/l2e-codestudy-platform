📖 Lesson: Functions and Conditions in Go

A function is a reusable block of code you can call by name:

func add(a int, b int) int {
    return a + b
}

func main() {
    fmt.Println(add(3, 5))  // prints: 8
}

An if/else condition lets your program make decisions:

number := 15
if number > 10 {
    fmt.Println("big")
} else {
    fmt.Println("small")
}

Chain multiple conditions with else if and use && for AND, || for OR:

if n%2 == 0 && n%3 == 0 {
    fmt.Println("divisible by both")
} else if n%2 == 0 {
    fmt.Println("divisible by 2 only")
} else if n%3 == 0 {
    fmt.Println("divisible by 3 only")
} else {
    fmt.Println("neither")
}

STEP 4 / 5 🎲 Random

Function with string condition — PrintIf

📖 How to do this: A function can take a string and return a different string based on a condition. This is directly from your fellowship checkpoint! 
func PrintIf(str string) string {
    if len(str) >= 3 || str == "" {
        return "G"
    }
    return "Invalid Input"
}

func main() {
    fmt.Println(PrintIf("hello"))  // prints: G
    fmt.Println(PrintIf("hi"))     // prints: Invalid Input
}
|| means OR — at least one condition must be true!

func FishAndChips(n int) string {

        return "error: number is negative"

        return "fish and chips"
    } else if n%2 == 0 {
        return "fish"
    } else if n%3 == 0 {
        return "chips"
    } else {
        return "error: non divisible"
    }
}
