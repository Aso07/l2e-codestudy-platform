📖 Lesson: Functions and Conditions in Go

A function is a reusable block of code you can call by name:

func add(a int, b int) int {
    return a + b
}
func main() {
    fmt.Println(add(3, 5))  // prints: 8
}

An if/else lets your program make decisions. Chain multiple with else if:

if n%2 == 0 && n%3 == 0 {
    fmt.Println("both")
} else if n%2 == 0 {
    fmt.Println("two only")
} else if n%3 == 0 {
    fmt.Println("three only")
} else {
    fmt.Println("neither")
}
