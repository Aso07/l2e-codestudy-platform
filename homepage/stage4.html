📖 Lesson: Functions and Conditions in Go

A function is a reusable block of code you can call by name. In Go, you define a function like this:

func add(a int, b int) int {
    return a + b
}

func main() {
    fmt.Println(add(3, 5))  // prints: 8
}

An if/else condition lets your program make decisions:

number := 15
if number > 10 {
    fmt.Println("big")
} else {
    fmt.Println("small")
}

You can chain multiple conditions with else if:

if number % 2 == 0 && number % 3 == 0 {
    fmt.Println("divisible by both")
} else if number % 2 == 0 {
    fmt.Println("divisible by 2 only")
} else if number % 3 == 0 {
    fmt.Println("divisible by 3 only")
} else {
    fmt.Println("not divisible by either")
}

Step 4 of 5 Function with string condition — PrintIf 🎲 Random

📖 How to do this: A function can take a string and return a string based on a condition. This is directly from your fellowship checkpoint! 
func PrintIf(str string) string {
    if len(str) >= 3 || str == "" {
        return "G"
    }
    return "Invalid Input"
}

func main() {
    fmt.Println(PrintIf("hello"))  // prints: G
    fmt.Println(PrintIf("hi"))     // prints: Invalid Input
}
|| means OR — at least one condition must be true!
