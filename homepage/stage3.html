📖 Lesson: Working with Strings in Go

A string in Go is any text wrapped in double quotes. You can do many useful things with strings:

// Get the length of a string
fmt.Println(len("hello"))    // prints: 5

// Strings have individual characters you can loop through
for _, ch := range "hello" {
    fmt.Println(string(ch))  // prints each letter
}

// Check if a string contains something
import "strings"
fmt.Println(strings.Contains("hello", "ell"))  // prints: true

The len() function counts how many characters are in a string. The range keyword lets you loop through each character one by one.

Step 2 of 5 Count a character in a string 🎲 Random

📖 How to do this: To count how many times a character appears, loop through the string and add 1 every time you find it: 
count := 0
for _, ch := range "hello world" {
    if ch == 'l' {
        count++
    }
}
fmt.Println(count)
Notice: single characters use single quotes 'l' not double quotes!

Step 4 of 5 Print the first word of a sentence 🎲 Random

📖 How to do this: The strings package can split a sentence into words using strings.Fields(). It splits on spaces and gives you a list of words: 
import "strings"

words := strings.Fields("hello world")
fmt.Println(words[0])  // prints: hello
words[0] means the first item — Go counts from 0!

Step 5 of 5 Check if a string contains a number 🎲 Random

📖 How to do this: To check if a string has a digit inside it, loop through each character and check if it is between '0' and '9': 
found := false
for _, ch := range "abc123" {
    if ch >= '0' && ch <= '9' {
        found = true
    }
}
fmt.Println(found)  // prints: true
