📖 Lesson: Working with Strings in Go

A string in Go is any text wrapped in double quotes. You can do many useful things with strings:

fmt.Println(len("hello"))    // prints: 5

for _, ch := range "hello" {
    fmt.Println(string(ch))
}

import "strings"
fmt.Println(strings.Contains("hello", "ell"))  // prints: true

The len() function counts characters. The range keyword loops through each one.

STEP 2 / 5 🎲 Random

Count a character in a string

📖 How to do this: Loop through the string and add 1 every time you find your character: 
count := 0
for _, ch := range "hello world" {
    if ch == 'l' {
        count++
    }
}
fmt.Println(count)
Single characters use single quotes 'l' not double quotes!

STEP 4 / 5 🎲 Random

Print the first word of a sentence

📖 How to do this: strings.Fields() splits a sentence into words. Use [0] to get the first: 
import "strings"
words := strings.Fields("hello world")
fmt.Println(words[0])  // prints: hello
[0] means first item — Go counts from 0!

STEP 5 / 5 🎲 Random

Check if a string contains a number

📖 How to do this: Loop through each character and check if it falls between '0' and '9': 
found := false
for _, ch := range "abc123" {
    if ch >= '0' && ch <= '9' {
        found = true
    }
}
