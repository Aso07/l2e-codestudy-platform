📖 Lesson: Numbers, Loops and Calculations in Go

In Go, you can print numbers directly with fmt.Println(). No quotes needed for numbers!

fmt.Println(42)       // prints: 42
fmt.Println(8 + 5)    // prints: 13
fmt.Println(20 / 4)   // prints: 5

To print a sequence of numbers, Go uses a for loop:

for i := 1; i <= 5; i++ {
    fmt.Println(i)
}

This prints 1, 2, 3, 4, 5 — each on its own line. The loop starts at 1, keeps going while i is less than or equal to 5, and adds 1 each time.

To check if a number is even or odd, use % which gives the remainder:

if i % 2 == 0 {
    fmt.Println(i)   // only prints even numbers
}

Step 3 of 5 Print even or odd numbers 🎲 Random

📖 How to do this: The % symbol gives you the remainder of a division. If a number divided by 2 has remainder 0, it is even. If the remainder is 1, it is odd. 
for i := 1; i <= 10; i++ {
    if i % 2 == 0 {
        fmt.Println(i)  // only even numbers
    }
}
To print odd numbers instead, change == 0 to != 0

Step 4 of 5 Print a calculation result 🎲 Random

📖 How to do this: In Go you can put a calculation directly inside fmt.Println() and it will calculate and print the result automatically. 
fmt.Println(8 + 5)   // prints 13
fmt.Println(4 * 6)   // prints 24
fmt.Println(20 / 4)  // prints 5
fmt.Println(20 - 7)  // prints 13
The symbols are: + add, - subtract, * multiply, / divide.

Step 5 of 5 Calculate a rectangle perimeter 🎲 Random

📖 How to do this: A variable in Go stores a value so you can use it later. The := symbol creates a new variable. 
w := 3           // create variable w with value 3
h := 4           // create variable h with value 4
p := 2 * (w + h) // calculate perimeter and store it
fmt.Println(p)   // print the result
The perimeter formula is: 2 × (width + height). Just change w and h to match your assigned values!
